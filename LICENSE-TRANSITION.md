# License Transition

`gmax` now ships under the Functional Source License 1.1 with the Apache 2.0
future-license variant (`FSL-1.1-ALv2`).

## Effective Date

This licensing change takes effect for this repository as of 2026-04-13.

## What This Means

- Current and future copies distributed with the new [LICENSE](LICENSE) file
  are source-available under `FSL-1.1-ALv2`.
- That license permits non-competing use, including internal use,
  non-commercial education, and non-commercial research.
- The same release automatically converts to Apache 2.0 on the second
  anniversary of the date it was made available.

## Earlier Apache-Licensed Copies

If you already received a copy of `gmax` under Apache License 2.0, that copy
remains available to you under the Apache 2.0 terms that shipped with it.
This transition changes the license for repository copies distributed with the
new license files; it does not retroactively revoke rights that were already
granted for older Apache-licensed copies.

## Canonical Files

- [LICENSE](LICENSE): current governing license text for newly distributed
  copies of this repository
- [NOTICE](NOTICE): concise operator-facing summary of the current license
  posture
